Enums

DiscordPermissions

Enum representing various Discord permissions.

  • CREATE_INSTANT_INVITE: Permission to create instant invites.
  • KICK_MEMBERS: Permission to kick members.
  • BAN_MEMBERS: Permission to ban members.
  • ADMINISTRATOR: Permission to be an administrator.
  • MANAGE_CHANNELS: Permission to manage channels.
  • MANAGE_GUILD: Permission to manage the guild.
  • ADD_REACTIONS: Permission to add reactions.
  • VIEW_AUDIT_LOG: Permission to view the audit log.
  • PRIORITY_SPEAKER: Permission to be a priority speaker.
  • STREAM: Permission to stream.
  • VIEW_CHANNEL: Permission to view channels.
  • SEND_MESSAGES: Permission to send messages.
  • SEND_TTS_MESSAGES: Permission to send TTS messages.
  • MANAGE_MESSAGES: Permission to manage messages.
  • EMBED_LINKS: Permission to embed links.
  • ATTACH_FILES: Permission to attach files.
  • READ_MESSAGE_HISTORY: Permission to read message history.
  • MENTION_EVERYONE: Permission to mention everyone.
  • USE_EXTERNAL_EMOJIS: Permission to use external emojis.
  • VIEW_GUILD_INSIGHTS: Permission to view guild insights.
  • CONNECT: Permission to connect.
  • SPEAK: Permission to speak.
  • MUTE_MEMBERS: Permission to mute members.
  • DEAFEN_MEMBERS: Permission to deafen members.
  • MOVE_MEMBERS: Permission to move members.
  • USE_VAD: Permission to use voice activity detection.
  • CHANGE_NICKNAME: Permission to change nickname.
  • MANAGE_NICKNAMES: Permission to manage nicknames.
  • MANAGE_ROLES: Permission to manage roles.
  • MANAGE_WEBHOOKS: Permission to manage webhooks.
  • MANAGE_EMOJIS_AND_STICKERS: Permission to manage emojis and stickers.
  • USE_APPLICATION_COMMANDS: Permission to use application commands.
  • REQUEST_TO_SPEAK: Permission to request to speak.
  • MANAGE_THREADS: Permission to manage threads.
  • USE_PUBLIC_THREADS: Permission to use public threads.
  • USE_PRIVATE_THREADS: Permission to use private threads.
  • USE_EXTERNAL_STICKERS: Permission to use external stickers.
  • SEND_MESSAGES_IN_THREADS: Permission to send messages in threads.
  • START_EMBEDDED_ACTIVITIES: Permission to start embedded activities.

Event

Enum representing various events in the application.

  • CommandNotFound: Event triggered when a command is not found.
  • ArgumentCastingError: Event triggered when there is an error casting an argument.
  • ExceptionDuringCommand: Event triggered when there is an exception during command execution.
  • CommandReceived: Event triggered when a command is received.
  • InvalidPermissions: Event triggered when a user has invalid permissions.

Exceptions

CommandNotFound

Exception raised when a command is not found.

ExceptionDuringCommand

Exception raised when there is an exception during command execution.

ArgumentCastingError

Exception raised when there is an error casting an argument.

InvalidPermissions

Exception raised when a user has invalid permissions.

Handler Class

The Handler class processes commands and events in a Discord bot.

__init__(self, app: Client, prefix: Union[str, List[str]], case_insensitive: bool = False)

Initialize the Handler.

  • app (Client): The Discord client instance.
  • prefix (Union[str, List[str]]): The prefix or list of prefixes for commands.
  • case_insensitive (bool): Whether command matching is case insensitive.

resolve_ping_to_role(self, guild: Guild, role_id_or_ping: Union[int, str])

Resolve a role mention or ID to a role ID.

  • guild (Guild): The guild in which to resolve the role.
  • role_id_or_ping (Union[int, str]): The role ID or mention.

resolve_ping_to_user(self, guild: Guild, user_id_or_ping: Union[int, str])

Resolve a user mention or ID to a user ID.

  • guild (Guild): The guild in which to resolve the user.
  • user_id_or_ping (Union[int, str]): The user ID or mention.

check_and_guess_param_types(self, func: Callable)

Check and guess parameter types for a command function.

  • func (Callable): The command function.

on_message(self, message: Message)

Handle an incoming message.

  • message (Message): The incoming message.

role_restricted(self, role: discord_Role)

Decorator to restrict a command to users with a specific role.

  • role (discord_Role): The required role.

user_restricted(self, user_id: list[int])

Decorator to restrict a command to specific users.

  • user_id (list[int]): List of allowed user IDs.

channel_restricted(self, channel_id: list[int])

Decorator to restrict a command to specific channels.

  • channel_id (list[int]): List of allowed channel IDs.

server_restricted(self, server_id: list[int])

Decorator to restrict a command to specific servers.

  • server_id (list[int]): List of allowed server IDs.

role_banned(self, role: discord_Role)

Decorator to ban a command from users with a specific role.

  • role (discord_Role): The banned role.

user_banned(self, user_id: list[int])

Decorator to ban a command from specific users.

  • user_id (list[int]): List of banned user IDs.

channel_banned(self, channel_id: list[int])

Decorator to ban a command from specific channels.

  • channel_id (list[int]): List of banned channel IDs.

server_banned(self, server_id: list[int])

Decorator to ban a command from specific servers.

  • server_id (list[int]): List of banned server IDs.
